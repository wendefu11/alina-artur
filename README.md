# Алина · Артур — Private Lounge

Личный игровой кабинет для двоих: статика на GitHub Pages, профили в localStorage, куча игр, дневник, статистика, романтика.

[Открыть сайт](#) (включи Pages — см. ниже)

## Что внутри

- **Два профиля** — Алина и Артур. Сохраняются на устройстве.
- **Главная** — счётчик дней вместе, цитата дня, настроение, быстрый старт.
- **Игры** (16 штук):
  - Вдвоём за устройством: крестики-нолики, Connect 4, камень-ножницы-бумага, Memory, Виселица, Pong.
  - Для пары: Колесо решений, Правда или Действие, Couple Quiz, Комплимент-машина, Монетка, Кубики, Love-O-Meter.
  - Соло (лучший счёт на профиль): Snake, 2048, Reaction.
- **Дневник** — записки и список желаний.
- **Статистика** — победы, поражения, рекорды по каждому профилю.
- **Три темы** — Aurora (тёмная романтика), Dawn (светлая), Noir (глубокая ночь).
- **Полностью статика** — никаких серверов. Работает с любого хостинга.

## Запустить локально

Любой статический сервер. Например:

```bash
python -m http.server 8000
# открой http://localhost:8000
```

## Развернуть на GitHub Pages

1. Запушь репозиторий на GitHub (см. ниже).
2. В настройках репозитория открой **Settings → Pages**.
3. **Source**: `Deploy from a branch`, **Branch**: `main`, **Folder**: `/ (root)`.
4. Подожди 1–2 минуты — сайт будет доступен на `https://<username>.github.io/<repo>/`.

Файл `.nojekyll` уже лежит в корне — Pages не будет ничего предобрабатывать.

## Старый сервер

В папке `server/` лежит исходная Python-версия с настоящим мультиплеером (комнаты, синхронизация). Запуск:

```bash
cd server
python alina.py
```

Если хочется играть по сети с разных устройств в реальном времени — используй её и подними где-нибудь (Render, Fly.io, Railway).

## Структура

```
index.html      — каркас SPA
styles.css      — все стили (aurora, glassmorphism, темы)
app.js          — роутер, главная, дневник, статистика, настройки
games.js        — все игры
storage.js      — обёртка над localStorage
data.js         — цитаты, правды, действия, вопросы для квиза
server/         — старый Python-сервер (мультиплеер)
```
