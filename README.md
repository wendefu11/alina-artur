# Алина и Артур · Private Lounge

Личный сайт для двоих. Тёмная-романтика + glassmorphism + 20 игр + дневник + статистика. Полностью статика — открывается с любого устройства.

**Сайт:** https://wendefu11.github.io/alina-artur/

## Что внутри

### Главная
- День-счётчик с настраиваемой датой («Вместе 651 день»).
- Цитата дня (меняется каждые сутки).
- Typewriter с любовными строками.
- Настроение Алины и Артура — один тап, другой видит сразу.
- Куплицовая сцена с двумя профилями и пульсирующим сердцем.

### Игры (20)

**Вдвоём за устройством (6):**
- Крестики-нолики, Connect 4, Камень-Ножницы-Бумага, Память (пары), Виселица, Pong.

**Соло с рекордами (6):**
- Snake, 2048, Реакция, Simon Says, Whack-a-Mole, 15-Puzzle, Minesweeper.

**Для пары (6):**
- Колесо решений, Правда / Действие, Машина комплиментов, Couple Quiz, Монетка, Кубики, Love-O-Meter.

### Дневник
- Личные записи на каждый профиль.
- Общий список желаний с шаблонами.

### Статистика
- Победы / поражения / ничьи / лучшая серия для каждого профиля.
- Рекорды по соло-играм.
- История последних 50 матчей.

### Настройки
- Два профиля: Алина, Артур.
- Три темы: Aurora (тёмная), Dawn (светлая), Noir.
- Экспорт / импорт всего стейта в JSON.
- Reset всех данных.

## Запустить локально

Любой статический сервер. Например:

```bash
cd alina
python -m http.server 8000
# открой http://localhost:8000
```

## Обновить деплой

```powershell
.\deploy.ps1
```

Или вручную:

```powershell
git add .
git commit -m "что обновилось"
git push
```

GitHub Pages автоматически перевыкатит сайт через 30–90 секунд.

## Структура

```
index.html      # SPA-каркас + login overlay + drawer
styles.css      # Все стили, 3 темы, анимации
app.js          # Router, home, diary, stats, settings
games.js        # 20 игр в одном модуле
data.js         # Цитаты, правды, действия, комплименты
storage.js      # LocalStorage с миграцией
.nojekyll       # GitHub Pages: без Jekyll
server/         # Старый Python multiplayer-сервер (отдельно)
```
